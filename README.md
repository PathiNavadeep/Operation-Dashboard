# OPS/25 – Operations Dashboard

OPS/25 is a modern, enterprise-grade Operations Dashboard designed to provide real-time operational visibility through a scalable, cache-first architecture. The system integrates 25 data sources, including public APIs, authenticated services, and scraper-based sources, to deliver actionable insights through interactive dashboards and automated workflows.

The application features a comprehensive collection of KPI cards, charts, tables, gauges, heatmaps, and analytics widgets, supported by an intelligent trigger engine that monitors predefined thresholds, generates SOP-driven actions, tracks SLA compliance, and manages an operational action queue. Every widget retrieves data through a centralized caching layer, ensuring consistent performance, efficient resource utilization, and a maintainable architecture.

## Features

* Enterprise-grade Operations Dashboard
* Cache-first architecture with configurable TTL-based refresh
* Integration with 25 public, authenticated, and scraper-based data sources
* Interactive KPI cards, charts, tables, gauges, heatmaps, and analytics widgets
* Intelligent SOP Trigger Engine for automated event detection
* SLA-based Action Queue with priority tracking and resolution workflow
* Role-based dashboard views for Founder and Analyst
* Global search and advanced widget filtering
* Responsive user interface with Dark and Light theme support
* Built using HTML, CSS, and JavaScript without external frameworks
* Designed for seamless integration with real backend services and APIs

This project demonstrates best practices in dashboard architecture, operational monitoring, workflow automation, and data visualization. Its modular design allows mock data sources to be replaced easily with production APIs, making it suitable as both a learning project and a foundation for enterprise-grade operational monitoring systems.
